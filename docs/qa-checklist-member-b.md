## ⚠️ 重要度の分類

フィードバック時に下記の絵文字または記号で重要度をつけてください：

- 🔴 **Critical（致命的）**：機能が動かない / クラッシュ / ナビゲーション不能 → **5/8 納品前に必ず修正**
- 🟡 **Major（重要）**：UXに大きな違和感 / 文言の意味が伝わらない → **修正推奨**
- 🟢 **Minor（軽微）**：見た目の細かいズレ / 表記ゆれ → **時間があれば修正**
- 💡 **Suggestion（提案）**：「こうしたら良くなりそう」というアイデア → **次フェーズ参考**
